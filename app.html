How It Works

Everything that makes Topset different from other trackers — explained.

📈
Strength / Bodyweight Ratios
Your Dashboard shows your key lifts as a bodyweight multiplier — for example, a 185lb lifter benching 200lbs is at 1.08× BW.

Tiers are based on Symmetric Strength general population standards, split by sex:

Beginner Intermediate Advanced Elite

The app scans your full log history for each lift and uses an Epley 1RM estimate from your best set — so you don't need to have maxed out to get a rating.
🧠
Fatigue-Aware Overload Hints
When you add an exercise to a session, Topset looks at the last time you logged that exercise and compares how many same-muscle exercises came before it then vs. now.

It then calculates a fatigue-adjusted target weight and rep count — not just a direction, but a specific number. Each extra same-muscle exercise beforehand applies a ~3.5% downward adjustment to the suggested weight; going in fresher bumps it up by the same amount, up to ±10%.

The hint shows your target for today prominently, with last session's numbers for reference. A blue delta means you're fresher than last time; orange means more pre-fatigued. The set weight fields are pre-filled with the adjusted target.
🏆
Fatigue-Adjusted PRs
Standard 1RM calculators treat every set equally. Topset doesn't.

For each PR candidate, the app counts how many prior working sets (at ≥85% of that weight) came before it. Each prior set applies a small upward adjustment — up to 10% total — to estimate what you could have done fresher.

A blue adj. badge means the displayed 1RM has been adjusted upward. Hover it to see the exact percentage. A grey raw badge means it was your first set — no adjustment needed.
✕
Logging Failed Sets
Add the letter f after a rep count to mark a set as failed — for example 8f means you attempted 8 reps but failed the last one.

Failed sets are excluded from PR calculations and 1RM estimates. They're still shown in your log so you have a complete record, but they won't inflate your numbers.

This is especially useful for powerlifting-style training where missed reps are meaningful data, not just noise.
🗓
Session-Based Program Cycles
Most apps count a "week" as 7 calendar days. Topset lets you count a cycle as completing a set of sessions — regardless of how many days it took.

For a Push / Pull / Legs split: once you've logged a Push, a Pull, and a Legs session in any order, Cycle 1 is complete and the app advances to Cycle 2 — adding the progression increment you set per block.

When you log a session, the app auto-detects which program day it matches by name. You can override this with the dropdown if needed.
📋
Workout Library & Programs
The Workout Library stores reusable workout templates — a list of exercises you do together. You can start any session from a template, or build it exercise by exercise.

Programs are built from your library workouts. Add workout blocks, set a progression rule per block (e.g. +5lbs/cycle on every exercise), and set the number of cycles. The program target replaces the overload hint for any exercise that's part of an active program.

Edit a workout in your library and the program automatically reflects the change — nothing goes stale.
📦
Your Data & Export
Everything is stored in your browser's local storage — no account, no server, no sync. Your data never leaves your device unless you export it.

Use ↑ Export in the header to copy a full JSON backup to your clipboard. Use ↓ Import to restore from a backup file. Do this regularly if you're relying on this as your primary log.

Clearing your browser data or using a different device will lose your log unless you've exported first.
⏱
Timers
Topset has two separate timers:

The Rest Timer is a countdown you start manually between sets. It floats above everything and stays running when you switch views. Tap the pill to expand presets (1m, 1:30, 2m, 3m, 5m).

The Session Timer starts automatically when you open a new session and counts up. It stops and saves the duration when you hit Save — visible on your session cards as "Duration".
